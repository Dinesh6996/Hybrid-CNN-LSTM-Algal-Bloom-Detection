A Hybrid CNN-LSTM Framework for Algal Bloom Detection in Water Quality Management
Project Overview

This project presents a Hybrid CNN-LSTM deep learning model for detecting algal blooms using water quality parameters. The system analyzes environmental data and predicts whether water is safe or unsafe based on the possibility of algal bloom formation.
